How to install via installer
---
1. Download the installer here and run it by clicking "run" if you are on Internet Explorer or Microsoft Edge. If you are on Chrome, click the downloaded file on the bottom instead.
2. If it asks for administrator priviledges and it shows "yes" or "no", click "yes".
3. You should see a screen like this. If so, click "OK".
4. You should now see a screen like this. If so, click "Next".
5. You should now see this. Press "I accept the agreement" and hit "Next".
6. When you see this screen, click "Next".
7. When you see this screen, also click "Next".
8. When this screen pops up, click "Create a desktop icon", then click "Next".
9. You should now see a screen that looks like this. If so, click "Install".
It will begin installing. It may take a minute, so let it complete. You can still do things on the coputer while it completes, as it will run in the background.
When it completes, you should see a screen like this. Uncheck "Launch Pulover's Macro Creator" and click "Finish".
10. Now, to launch the program, go to your desktop. There should be a new icon on your desktop that looks like this. Double-click it to launch it.
11. After a couple seconds, the program should launch and you will see this. Hit "OK" on the message that pops up.
12. You should now see this. If so, go to "File --> New" and click "New" to create a new macro.
13. You should now see this.
14. Click on the button I have circled in red. This should now show up. Click "OK".
15. You should now see this. Click on the button I have circled in red.
16. You should see a popup like this. Type in "4", click "Minutes", then click "OK".
It should now look like this.
Now, drag the text "[Pause] in middle of the two loops. You should now see this.
Then, click the button I circled in red.
You should then see this.
Type in a dot (.), then click "OK".
You should then see this.
Now, you can hit the button I circled in red to play the macro. Every 4 minutes, a dot (.) will be typed in, thus preventing your computer from entering sleep mode.
To stop the macro, click the button circled in red,
then a menu like this should pop up. Right-click the icon I circled in red, and click "Exit". If a message pops up about saving a project, click "No".
For the steps on only how to create the macro, click here.
